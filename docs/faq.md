# 常见问题 FAQ

这里整理了 API 中转平台在接入、计费、兼容性、分组和模型选择方面的常见问题。

> API 平台地址：<https://jeniya.cn>

## 1. 什么是 API 中转站？

API 中转站用于统一接入多个大模型 API，帮助开发者更方便地调用 GPT、Claude、Gemini、Grok 和国产模型等能力。

## 2. 平台支持哪些模型？

平台支持 GPT、Claude、Gemini、Grok、DeepSeek、Qwen 以及其他部分模型，具体以平台后台展示为准。

你也可以查看：

- [支持模型列表](/models.html)

## 3. 是否兼容 OpenAI 格式？

是。平台默认兼容 OpenAI 格式，便于开发者使用现有 SDK 和代码快速接入。

## 4. 如何收费？

平台主要有两种计费方式：

- 文本模型通常按 Tokens 收费
- 图像模型通常按调用次数收费

不同模型和分组的费率可能不同，具体以后台实时展示为准。

你也可以查看：

- [价格说明](/pricing.html)

## 5. 为什么会有分组？

分组主要用于区分不同上游渠道、模型来源、可用性、价格和适用场景，便于用户根据稳定性、能力和预算进行选择。

## 6. default 默认分组适合什么用户？

default 默认分组适合大多数综合型需求用户，一般支持 ChatGPT、Claude、MJ 和部分国产模型。

## 7. 企业级高可用大模型分组适合什么场景？

适合对稳定性和连续可用性要求较高的业务，尤其适用于国产模型相关场景。

## 8. 优质gemini 和官转gemini 有什么区别？

两者都支持 Gemini 模型，但官转gemini 通常资源更多、成本更高，因此费率也更高，更适合对可用性要求更高的用户。

## 9. 图片模型怎么收费？

图片模型通常按调用次数收费，例如 GPT Image 和 Gemini 的部分图像能力，实际价格请以平台后台为准。

## 10. 为什么不同模型价格差别很大？

不同模型在能力、上下文长度、速度、稳定性、上游资源和供应渠道方面存在差异，因此价格会不同。

## 11. 为什么不同分组费率不一样？

因为不同分组的上游渠道、账号资源、运维成本和可用性保障不同，所以费率会不同。

## 12. 我应该选哪个模型？

可以按场景简单理解：

- **通用任务**：GPT 系列
- **长文本和复杂写作**：Claude 系列
- **多模态与图像相关任务**：Gemini 系列
- **中文和成本优先**：国产模型
- **特定实验型能力**：Grok 系列

## 15. 如果报 401 / 429 / 超时怎么办？

建议先检查：

- API Key 是否正确
- 请求格式是否符合 OpenAI 兼容规范
- 模型名称是否正确
- 当前账户余额或额度是否充足
- 请求频率是否过高

你也可以阅读相关专题内容：

- [GPT API 专题](/blog/gpt-api/)
- [Claude API 专题](/blog/claude-api/)
- [Gemini API 专题](/blog/gemini-api/)

## 16. 图像模型和文本模型有什么区别？

文本模型通常用于：

- 问答
- 写作
- 翻译
- 代码生成

图像模型通常用于：

- AI 生图
- 海报生成
- 封面图生成
- 图片创作辅助

## 17. 是否适合企业接入？

如果你需要多模型统一接入、成本控制、快速切换模型或更适合国内开发环境的方案，那么 API 中转平台非常适合作为企业接入方案之一。

## 18. 我可以直接用 OpenAI SDK 吗？

一般情况下可以。因为平台兼容 OpenAI 格式，所以多数情况下可以直接复用 OpenAI SDK 和现有调用逻辑。
